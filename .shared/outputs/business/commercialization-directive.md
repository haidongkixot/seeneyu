# Seeneyu Commercialization Directive
## From: Business Development Director, PeeTeeAI
## Date: 2026-03-30
## To: PM of Seeneyu → Development Team

---

## Executive Summary

After a thorough product audit, Seeneyu scores **5/10 for commercialization readiness**. The core product is strong, but we're **giving away 85% of the value for free**. Users have zero incentive to upgrade because they can complete the entire Foundation curriculum, access all 65+ clips, play unlimited mini-games, and chat with Coach Ney 5x/day — all without paying.

**We need to create strategic friction that makes users WANT to pay, not HAVE to pay.**

---

## TIER RESTRUCTURE — "The Value Ladder"

### FREE (Explorer) — Taste the magic
| Feature | Limit | Purpose |
|---------|-------|---------|
| Foundation Lessons | First 2 per course only | Hook them, leave them wanting more |
| Library Clips | Beginner difficulty only (~20 clips) | Show quality, gate depth |
| Practice Recording | 15 seconds max | Enough to try, short enough to frustrate |
| AI Feedback | Score only (no details) | Show the number, hide the insights |
| Coach Ney | 3 messages/day, text only | Taste the coach, can't have a real conversation |
| Arcade | 1 challenge per type | Sample, not feast |
| Mini-Games | 2 games only (Guess + Match) | Fun hook, gate the premium games |
| Hearts | 3/day, no freeze | Creates daily ceiling |
| Gamification | XP, badges, streak | Keep engagement loop even on free |
| Discussions | Read only, can't post | See community value, must pay to join |

### STANDARD ($12/month, $96/year) — Serious learner
| Feature | Limit | Purpose |
|---------|-------|---------|
| Foundation Lessons | All lessons, all courses | Full curriculum access |
| Library Clips | Beginner + Intermediate (~50 clips) | Majority of content |
| Practice Recording | 60 seconds | Proper practice time |
| AI Feedback | Full detailed analysis (4 scores + tips) | The real value proposition |
| Coach Ney | 20 messages/day + voice | Real coaching conversations |
| Arcade | Unlimited, all difficulties | Full practice playground |
| Mini-Games | All 5 games | Complete game suite |
| Hearts | 10/day + 1 freeze/week | Generous but bounded |
| Gamification | Full (leaderboards, tiers, daily quests) | Social competition |
| Discussions | Full read + write | Community participation |
| Weekly Report | Email digest | Engagement touchpoint |
| Learning Plan | Personalized daily plan | Coach Ney proactive |

### ADVANCED ($24/month, $192/year) — Master communicator
| Feature | Everything in Standard PLUS | Purpose |
|---------|---------------------------|---------|
| Library Clips | All difficulties (65+ clips) | Complete library |
| Practice Recording | 3 minutes | Full scene reenactment |
| AI Feedback | Full + advanced muscle-level analysis | Premium coaching depth |
| Coach Ney | Unlimited + voice + priority | Dedicated AI coach |
| VIP Content | 12 exclusive masterclass lessons | Content not available elsewhere |
| Expression King Pro | All challenges + certificate | Certification value |
| Hearts | Unlimited | No friction |
| Priority Support | 24hr email response | Peace of mind |
| Analytics Dashboard | Personal progress charts | Data-driven improvement |
| Ad-free | No upsell prompts | Premium experience |

## CRITICAL FIXES — Priority Order

### P0: CONVERSION KILLERS (Do this week)

**1. Remove admin approval from sign-up**
- Auto-approve all learner accounts immediately
- Keep approval only for admin role
- Users must start learning within 30 seconds of signing up

**2. Gate Foundation Lessons**
- Free: Lessons 1-2 per course (the hook)
- Standard: All lessons
- Show lock icon on lesson 3+ with "Upgrade to continue learning"
- After completing lesson 2, show upgrade modal with 7-day free trial offer

**3. Gate Library by Difficulty**
- Free: Beginner only
- Standard: Beginner + Intermediate
- Advanced: All
- Show blurred thumbnails for locked clips with plan badge

**4. Upgrade Prompts at Friction Points**
- When user hits recording limit: "Want longer recordings? Upgrade →"
- When user hits Coach Ney limit: "Coach Ney has more to say. Unlock unlimited →"
- When user sees locked lesson: "This lesson continues your journey. Start 7-day trial →"
- When hearts run out: "Out of hearts! Upgrade for unlimited practice →"
- Style: Coach Ney avatar + speech bubble (not generic modal)

### P1: VALUE PERCEPTION (Do in 2 weeks)

**5. Redesign Pricing Page**
- Show outcomes, not features: "Get 10x more feedback detail" not "Full analysis"
- Add social proof: user count, testimonials from paid users
- Add FAQ section
- Add 7-day free trial for Standard
- Add money-back guarantee badge

**6. Upgrade Onboarding Flow**
- After skill assessment, show: "Based on your level, Standard plan will help you improve 2.5x faster"
- Add goal question: "What's your #1 communication goal?"
- Show personalized content preview: "Here's what Standard unlocks for your eye contact weakness"
- Offer 7-day free trial immediately

**7. Homepage Upgrade Section**
- Between testimonials and CTA, add "Choose Your Path" section
- Show 3 plans with key differentiator highlighted
- Visual: progression from Free → Standard → Advanced

**8. Define "VIP Content" for Advanced**
- Create 12 exclusive masterclass-style lessons
- Topics: "The CEO Stance", "Negotiation Body Language", "First Impression Science"
- Available ONLY to Advanced tier (never free)
- This is the #1 reason to choose Advanced over Standard

### P2: MONETIZATION INFRASTRUCTURE (Do in 4 weeks)

**9. Free Trial System**
- 7-day free trial for Standard plan
- Credit card required upfront
- Auto-converts to paid after 7 days
- Cancellation: 1-click during trial
- Track trial-to-paid conversion rate

**10. Upgrade Prompt Intelligence**
- Track which limits users hit most frequently
- Personalize upgrade message based on what they're blocked on
- A/B test upgrade prompt copy and timing
- Coach Ney suggests upgrade naturally in conversation

**11. Coupon/Promotion System**
- Create `Coupon` model: code, discount%, validUntil, maxUses
- Launch offer: "EARLYBIRD" — 50% off first 3 months
- Referral program: "Give $5, Get $5"
- Seasonal campaigns

**12. Cancellation Flow + Win-Back**
- Exit survey: "Why are you leaving?"
- Offer pause (1 month freeze) instead of cancel
- Win-back email sequence: Day 1, Day 3, Day 7 after cancellation
- Special "come back" offer: 30% off for 2 months

### P3: GROWTH MECHANICS (Do in 6 weeks)

**13. Referral System**
- Each user gets unique referral link
- Referred user gets: 7-day free Standard trial
- Referrer gets: 1 month free when referral converts to paid
- Leaderboard: "Top Referrers" (gamify referrals)

**14. Embeddable Mini-Games for Viral Growth**
- Games can be embedded on any website via iframe
- Game shows seeneyu branding + "Play more at seeneyu.com"
- Score submission requires sign-up (lead capture)
- Target: productivity blogs, communication training sites, LinkedIn articles

**15. B2B Team Plans**
- Team plan: $8/user/month (min 5 users)
- Admin dashboard: see team progress, assign lessons
- Manager reports: weekly team improvement metrics
- Enterprise: custom pricing, SSO, dedicated support

## UX/UI IMPROVEMENTS FOR CONVERSION

### Homepage
- Hero: Add "Free trial" badge next to CTA
- Add "Trusted by X learners" counter (even if small)
- Add "As seen on" logos (if applicable) or "Built with AI from OpenAI"
- Add mini video demo (30s showing the practice flow)

### Pricing Page
- Make Standard visually dominant (larger card, "Most Popular" badge)
- Add toggle animation for monthly/annual
- Show savings: "Save $24/year with annual billing"
- Add feature comparison accordion (expandable)
- Add "Questions?" chat with Coach Ney on pricing page

### Locked Content UI
- Don't hide locked content — show it blurred/dimmed with lock icon
- "Unlock this lesson" overlay with 1-click trial start
- Preview first 30 seconds of locked videos
- Show what you'll learn (bullet points) even when locked

### Upgrade Modals
- Use Coach Ney avatar (familiar, trustworthy)
- Show specific benefit: "Unlock this lesson" not "Upgrade your plan"
- Include "7-day free trial" button prominently
- "No thanks, I'll stay on Free" link (less prominent, gray)

### Mobile App
- Fetch plans from API (not hardcoded)
- Deep link back from web checkout
- Show upgrade prompts at same friction points as web
- In-app purchase via RevenueCat (future)
